# LaunchGrid - Getting Started Guide

Welcome to LaunchGrid! This guide will walk you through everything you need to know to plan, manage, and report on your marketing campaigns.

---

## What is LaunchGrid?

LaunchGrid is a marketing campaign planning platform that replaces spreadsheets with an interactive, visual interface. You can plan campaigns across multiple channels (Social, Email, Paid Ads, Events, etc.), track budgets, manage events, and generate executive reports — all from one place.

---

## 1. Your First Steps

### Creating a Workspace

A **workspace** is your planning hub — think of it as one calendar for a team, quarter, or region.

1. Click **"New Workspace"** from the home screen
2. Give it a name (e.g., "Q2 2026 Marketing Plan")
3. You're in! You'll land on the **Timeline view**

### Setting Up Swimlanes (Channels)

Swimlanes are the rows on your timeline. Each one represents a marketing channel.

1. Click **"+ Add Swimlane"** on the left sidebar
2. Name it (e.g., "Email", "Paid Ads", "Social", "Events", "Content", "PR")
3. Optionally set a budget for each swimlane
4. Repeat for each channel you want to track

### Defining Status Labels

Statuses help you track where each activity stands.

1. Go to workspace **Settings**
2. Under **Statuses**, add labels like:
   - **Considering** (grey)
   - **Negotiating** (yellow)
   - **Committed** (green)
3. Assign colors so they're easy to spot at a glance

---

## 2. Creating Activities

Activities are the individual items on your plan — a blog post, an ad campaign, an email send, etc.

### Option A: Drag on the Timeline
- Click and drag across a date range in any swimlane row
- A new activity is created instantly for those dates
- Click it to add details

### Option B: Click on the Calendar
- Switch to **Calendar view**
- Click on any empty day to create a new activity

### Option C: Use the Activity Form
- Click the **"+ New Activity"** button
- Fill in the details in the modal that appears

### Option D: AI Brief Generator
- Click **"Generate from Brief"**
- Paste your campaign brief or description
- LaunchGrid's AI will automatically create a structured set of activities with suggested dates, swimlanes, and cost estimates

### What You Can Add to an Activity

| Field | What It's For |
|-------|--------------|
| **Title** | Name of the activity |
| **Dates** | Start and end date |
| **Swimlane** | Which channel it belongs to |
| **Campaign** | Group related activities together |
| **Status** | Current stage (Considering, Committed, etc.) |
| **Cost** | Planned and actual spend |
| **Currency** | USD, GBP, or EUR |
| **Region** | US, EMEA, or ROW |
| **Tags** | Custom labels for filtering |
| **Description** | Details, notes, and context |
| **Attachments** | Files from Google Drive |
| **Slack Channel** | Link to a Slack channel for updates |

## 3. Views — Different Ways to See Your Plan

LaunchGrid offers multiple views. Switch between them using the tabs at the top.

### Timeline View (Default)
- Gantt-style chart with drag-and-drop
- **Drag an activity** left/right to change dates
- **Drag the edges** to make it longer or shorter
- **Drag up/down** to move it to a different swimlane
- **Zoom levels**: Year, Quarter, or Month for different levels of detail
- A **today line** shows where you are

### Calendar View
- Traditional month grid
- Activities appear as colored blocks spanning their date range
- Great for seeing what's happening in a specific week or month

### Table View
- Spreadsheet-style list of all activities
- Sort by any column (title, dates, status, cost, etc.)
- Edit fields directly in the table
- Show/hide columns to focus on what matters

### Dashboard
- Visual summary of your budget, pipeline, and performance
- **Budget tracking**: Planned vs. actual spend by channel, campaign, or region
- **SAO metrics**: Expected vs. actual Sales Accepted Opportunities
- **Pipeline & revenue**: How much pipeline and revenue each activity generates
- **Variance alerts**: See which campaigns are over or under budget

## 4. Managing Events

For conferences, trade shows, and webinars, LaunchGrid has a dedicated **Events** section.

### Creating an Event
1. Go to the **Events** tab
2. Click **"+ New Event"**
3. Fill in: name, dates, location, and venue

### Adding Sub-Events
Under any event, add sessions like:
- Workshops
- Customer dinners
- Product demos
Each with its own time, location, and attendee list.

### Managing Attendees
- Add **internal attendees** (your team) and **customer attendees**
- Track **travel status** for each person
- Manage **pass allocation** (who has a badge/ticket)

### Event Checklist
- Add tasks for event readiness (e.g., "Book venue", "Send invites", "Print badges")
- Check them off as you go
- See completion percentage at a glance

### Calendar Invites
- Generate **.ics calendar invites** and distribute to attendees
- Invites include event details, times, and location

### Slack Notifications
- Link your event to a Slack channel
- Send status updates directly to the team

## 5. Campaigns — Grouping Activities Together

A **Campaign** groups related activities across different channels.

For example, a "Spring Product Launch" campaign might include:
- A blog post (Content swimlane)
- A paid ad series (Paid Ads swimlane)
- An email nurture sequence (Email swimlane)
- A webinar (Events swimlane)

### Creating a Campaign
1. Go to **Campaigns** in the sidebar
2. Click **"+ New Campaign"**
3. Name it and set an overall budget
4. When creating activities, assign them to this campaign

This lets you track total spend and performance across all channels for one initiative.

## 6. AI Features

### Copilot (Ask Questions)
Click the **Copilot** icon to ask questions in plain English:
- "How much have we spent on the Spring Launch campaign?"
- "Which campaigns are over budget?"
- "What is the ROI for our EMEA events?"
- "Compare US vs EMEA spending"
- "Top performing campaigns this quarter"

You'll get an answer plus a supporting data table.

### Brief Generator
Paste a campaign brief and let AI do the heavy lifting:
1. Click **"Generate from Brief"**
2. Paste your campaign description
3. AI creates activities with dates, swimlanes, cost estimates, and descriptions
4. Review, adjust, and save

Supported campaign types: Product Launches, Use Cases, Themes, Expansions, Partnerships, Customer Success, Demand Gen, ABM, and Community campaigns.

## 7. Reports & Exports

### PowerPoint Reports
Generate professional decks with one click:
- **Campaign Performance Deck** — Metrics, trends, and AI insights
- **Budget Review Deck** — Spend analysis and variance reports
- **Event ROI Deck** — Event performance and attendee analysis
- **Campaign Details Report** — Granular campaign breakdown

Reports include AI-generated strategic insights automatically.

### Other Exports
- **PNG** — Screenshot any view as an image
- **CSV** — Export activity data to a spreadsheet

## Quick Reference — Common Tasks

| I want to... | Here's how |
|--------------|-----------|
| Create an activity | Drag on timeline, click calendar date, or use "+ New Activity" |
| Reschedule something | Drag it left/right on the timeline |
| Change an activity's channel | Drag it up/down to a different swimlane |
| See budget status | Go to the Dashboard tab |
| Generate a report | Go to Reports and select a deck type |
| Ask a question about my data | Open Copilot and type your question |
| Auto-create activities from a brief | Use "Generate from Brief" and paste your text |
| Manage event attendees | Go to Events > select event > Attendees tab |
| Export my plan | Use the export button for PNG, CSV, or PPTX |
| Link to Slack | Edit an activity and add a Slack channel link |

## Tips for Success

1. **Start with swimlanes** — Set up your channels before adding activities
2. **Use campaigns to group** — It makes budget tracking and reporting much easier
3. **Drag and drop is your friend** — The timeline is built for quick rearranging
4. **Check the dashboard regularly** — Stay on top of budget and performance
5. **Use the AI Brief Generator** — Save time when planning new campaigns
6. **Generate reports before meetings** — One-click decks with AI insights included
